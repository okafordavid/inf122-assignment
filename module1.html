Unit 1

Definitions of Web Programming, Internet and WWW

Web Programming is the process of creating websites and web applications using technologies such as HTML, CSS and JavaScript.

The Internet is a worldwide network that connects computers and devices for communication and information sharing.

The World Wide Web (WWW) is a collection of web pages accessed through web browsers over the Internet.
